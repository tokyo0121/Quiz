### 作品説明

「オフライン」と「オンライン」。
正反対の言葉のように思えるけど、実はたったの１文字違い。

その小さな「差」を、人と人との距離を埋められる、とっておきのアプリケーションです。


### 特徴

- 特徴１
  アイスブレイクに最適
  
  
- 特徴２
  並外れた着眼点とデザイン力
